# samOS RootFS Information

This folder contains documentation related to the **samOS root filesystem**.  
It does not include the full rootfs image itself — the complete compressed rootfs (`tar.xz`) is available for download in the **Releases** section of this repository.

## About samOS
- **Base Distribution:** Arch Linux  
- **Architecture:** AMD64  
- **Purpose:** Lightweight and Fast operating system built on Arch Linux principles.  

## Contents
- `packages-list` → A list of installed packages included in the rootfs info.  
- Additional notes and metadata about the rootfs build process.  

## Notes
- The **rootfs info folder** is intended only for transparency and documentation of installed packages and related information.  
- The **full rootfs image** is published separately under [Releases](../../releases).  
- Users can review the package list here to understand the composition of the system before downloading the full image.
